# 项目 10：综合 CTF 毕业挑战

> 对应章节：贯穿 Ch01 ~ Ch12

## 目标

独立完成一次包含 **Web / Pwn / Reverse / Crypto / Forensics / OSINT / Misc** 的综合 CTF 训练，验证你是否已经把前 9 个项目学到的技能融会贯通。

不设标准答案，以"能不能独立做出、能不能把思路讲明白"作为毕业衡量。

## 两条路线（任选其一，或都做）

### 路线 A：参加一场线上 CTF

- **PicoCTF**（常年开放）：适合入门 → 中级
- **0CTF / TCTF**、**XCTF 分站赛**、**强网杯** 的题目（官方公布过去题）
- **RITSEC / UTCTF / DefCamp / ICTF** 等国际赛事的 past CTFs

目标：做出 **≥ 15 题**，覆盖所有分类。

### 路线 B：渗透一个综合靶场

- **HackTheBox**：完成 1 条 Active Directory Pro Lab（推荐 Dante / RastaLabs 精简版）或 10 台独立机器（不同难度）。
- **TryHackMe**：完成 `Throwback`、`Zero Logon`、`Offensive Pentesting` 路径之一。
- **VulnHub**：挑选 5 个"Hard"难度的靶机连打。

## 统一要求

不管走哪条路，你需要产出：

### 1. 攻击日志 + 报告

- 按题目 / 靶机分类，保留：
  - 提示 / 题目文件
  - 侦察命令与输出
  - 关键思路（特别是卡壳之后怎么突破的）
  - 完整 exp / payload / wordlist
  - 最终 flag / root 凭证

### 2. ATT&CK 映射

对路线 B，至少映射出 10 个 ATT&CK 技术（TID）到你的攻击链。

### 3. 防御建议

对每个题目，至少写一条"如果我是蓝队怎么发现 / 阻止"。

## 推荐工作流

```
┌─── Recon 阶段 ──────────────────┐
│ nmap / masscan / gobuff / ffuf  │
│ wappalyzer / whatweb / nuclei   │
└────────────┬────────────────────┘
             ▼
┌─── Enumeration ────────────────┐
│ Web 目录 / 子域 / 敏感文件      │
│ SMB share / SNMP / LDAP / NFS  │
└────────────┬────────────────────┘
             ▼
┌─── Exploitation ───────────────┐
│ CVE / 业务逻辑 / 弱口令 / 0day  │
└────────────┬────────────────────┘
             ▼
┌─── Foothold → User ────────────┐
│ 反弹 shell / 稳定 shell         │
│ tty 升级 (script -qc /bin/bash) │
└────────────┬────────────────────┘
             ▼
┌─── Privilege Escalation ───────┐
│ linpeas / winpeas / gtfobins    │
└────────────┬────────────────────┘
             ▼
┌─── Loot / Pivot ───────────────┐
│ 凭证 / 内网映射 / 代理(chisel)  │
└────────────┬────────────────────┘
             ▼
┌─── Report ─────────────────────┐
│ writeup + 防御建议              │
└────────────────────────────────┘
```

## 交付物

- `writeup/`：你的 WriteUp（可上传到自己的博客 / GitHub Pages）
- `exploits/`：所有自写脚本
- `final_report.md`：总报告（建议 ≥ 3000 字）
- （可选）录制 1 段 20 分钟以内的视频 WriteUp

## 毕业标准（给自己打分）

| 项 | 满分 | 自评 |
|----|------|------|
| 独立解题数量（≥15 / ≥3 box） | 30 | |
| 覆盖的题型分类数 | 20 | |
| WriteUp 质量（逻辑 / 图示 / 复现步骤） | 20 | |
| ATT&CK 映射清晰度 | 10 | |
| 防御建议实用性 | 10 | |
| 时间线 / 攻击链复原度 | 10 | |

≥ 80 分即视为毕业，可以开始投 **安全工程师 / 渗透测试 / 红队 / 蓝队** 相关实习 / 岗位简历。
