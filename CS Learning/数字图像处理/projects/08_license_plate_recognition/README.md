# 项目 08：车牌识别 (综合实战)

> 对应章节：Ch 3（增强）+ Ch 6（颜色）+ Ch 9（形态学）+ Ch 10（分割）+ Ch 11（特征）
> 目标：走一遍传统图像处理完整工程链路：**定位 → 矫正 → 字符分割**（OCR 可选）。

## 流程

```
原图
 │ HSV 蓝色/黄色 mask
 ▼
颜色候选区
 │ 形态学 (Close / Open) + 长宽比过滤
 ▼
矩形车牌框 ──→ 仿射/透视校正
 │ OTSU 二值
 ▼
字符连通域 → 分割框
 │ (可选) CNN / pytesseract
 ▼
识别结果
```

## 包含内容

- HSV 颜色分割（蓝牌/黄牌）
- 形态学闭运算连接文字、开运算去噪
- 轮廓检测 + 长宽比筛选
- `minAreaRect` 得到斜矩形 → `warpPerspective` 校正
- 字符分割：投影 + 连通域
- （可选）pytesseract OCR，若未安装会跳过

## 运行

```bash
python main.py --image assets/car.jpg
```

若未提供图，脚本会生成**合成车牌图**用于演示流水线。

## 注意

- 真实复杂场景（夜景、遮挡、倾斜 > 30°）需要端到端深度模型（YOLO + CRNN），本项目仅演示传统方法。
- 想获得真正可用的识别率，可替换为 HyperLPR、EasyPR、PaddleOCR 等。
