# 项目 03：频率域滤波

> 对应章节：第 4 章
> 目标：掌握 FFT 可视化，设计 ILPF / BLPF / GLPF 低通、对应高通，以及陷波去除周期噪声。

## 包含内容

- FFT 频谱与中心化
- 理想 / Butterworth / Gaussian 低通
- 对应高通（锐化）
- 同态滤波（简化版）
- **陷波滤波**：合成带周期条纹的图像并恢复

## 运行

```bash
python main.py
python main.py --image my.jpg
```

## 输出

- 频谱可视化
- 三种低通对比（观察 ILPF 的振铃）
- 高通锐化效果
- 周期噪声合成 → 陷波恢复

## 延伸题目

- 调整 D0 看过渡带与振铃变化。
- 对真实扫描图中的摩尔纹做陷波：手动定位频谱中的对称亮斑。
