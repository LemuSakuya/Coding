# 单片机开发项目清单

## 01_key_led_flow

按键控制流水灯实验（与你当前课程作业匹配）：
- K1：8 路 LED 正向流水
- K2：8 路 LED 反向流水
- K3：高 4 位 / 低 4 位交替闪烁
- K4：8 路 LED 全闪
- K5（扩展）：LED0&7 -> LED1&6 -> LED2&5 -> LED3&4 循环

## 02_timer_interrupt_counter

定时器中断实验：
- 定时器 T0 1ms 中断
- 软件计数得到 1 秒节拍
- LED 每 1 秒翻转一次

## 说明

每个项目目录均包含：
- README.md：实验要求、连线说明、运行步骤
- main.c：可直接放入 Keil C51 工程的参考代码
