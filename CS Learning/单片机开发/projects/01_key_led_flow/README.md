# 实验 01：按键控制流水灯

## 一、实验目标

基于 51 单片机完成按键控制 8 路 LED 的显示模式切换，并实现 1 个扩展功能。

## 二、硬件连接约定

- 单片机：AT89C51（或 STC89C52）
- LED0~LED7：接 P3.0~P3.7（低电平点亮）
- K1~K5：接 P1.0~P1.4（按下为低电平）

## 三、功能要求

1. 按下 K1：LED 正向流水（P3.0 -> P3.7）
2. 按下 K2：LED 反向流水（P3.7 -> P3.0）
3. 按下 K3：高 4 位与低 4 位交替闪烁
4. 按下 K4：8 路 LED 全闪
5. 按下 K5（扩展）：
   - LED0 和 LED7 同时亮 500ms
   - LED1 和 LED6 同时亮 500ms
   - LED2 和 LED5 同时亮 500ms
   - LED3 和 LED4 同时亮 500ms
   - 循环执行

## 四、程序说明

- 按键扫描带软件消抖（10ms）
- 主循环根据当前 mode 调用对应显示函数
- 每个显示函数中都包含“中途可切换模式”的快速退出判断

## 五、运行步骤

1. 在 Keil 新建 C51 工程，添加 main.c
2. 编译生成 hex 文件
3. 在 Proteus 中加载 hex，完成按键和 LED 接线
4. 运行仿真并按键验证 5 种模式
