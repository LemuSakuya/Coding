# 实验报告可直接参考的文字（实验二）

## 1. 实验原理

数码管显示由段选与位选共同完成。段选决定当前显示的数字形状，位选决定当前点亮哪一位数码管。多位显示通常采用动态扫描方式，通过快速轮流点亮每一位实现“同时显示”的视觉效果。

## 2. 静态显示结果

将第 1 位数码管固定选通，段码按 0~9 依次输出，每 500ms 更新一次，能够稳定观察到单个数码管循环显示 0~9。

## 3. 动态显示结果

使用定时器 T0 1ms 中断，在中断中执行位扫描，主循环周期更新显示数字。4 位数码管可稳定显示递增数字（0000~9999），无明显闪烁，说明刷新频率满足视觉暂留要求。

## 4. 遇到的问题与解决

1. 现象：数码管全亮或全灭。
   - 原因：共阳/共阴器件与程序电平逻辑不匹配。
   - 解决：调整 `SEG_ACTIVE_LEVEL` 与 `DIGIT_ACTIVE_LEVEL` 宏。
2. 现象：动态显示重影。
   - 原因：切换位选前未先关闭全部位，或刷新节拍不稳定。
   - 解决：扫描时执行“灭位 -> 写段码 -> 开位”，并放入定时中断节拍。

## 5. 结论

本实验完成了数码管静态显示和动态扫描显示，掌握了段码映射、位扫描时序与定时器中断驱动显示的方法，为后续矩阵键盘与人机交互实验打下基础。
