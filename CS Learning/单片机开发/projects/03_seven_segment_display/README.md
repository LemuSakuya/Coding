# 实验 03：数码管显示（静态 + 动态）

> 对应课程作业里的“实验二：数码管显示实验”。

## 一、实验目标

1. 理解 7 段数码管显示原理（段选 + 位选）
2. 掌握单个数码管静态显示 0~9
3. 掌握 4 位数码管动态扫描显示
4. 学会使用定时器中断实现稳定刷新

## 二、硬件连接约定

- 单片机：AT89C51 / AT89C52（12MHz）
- 段选：P0.0~P0.7 -> A B C D E F G DP（建议串电阻网络）
- 位选：P2.0~P2.3 -> DIG1 DIG2 DIG3 DIG4

如果你在 Proteus 里选择了共阳/共阴不同器件，电平极性要改：

- `SEG_ACTIVE_LEVEL`：段选有效电平
- `DIGIT_ACTIVE_LEVEL`：位选有效电平

这两个宏在 `main.c` 顶部。

## 三、功能说明

代码里提供两种模式：

- `MODE_STATIC_SINGLE_DIGIT`：静态显示（单个数码管循环 0~9）
- `MODE_DYNAMIC_FOUR_DIGIT`：动态显示（4 位数码管显示 0000~9999）

切换方式：修改 `EXPERIMENT_MODE`。

## 四、核心实现思路

1. 建立段码表 `seg_lut_raw`
2. 静态模式：固定选通第 1 位，循环输出段码
3. 动态模式：
   - T0 配成 1ms 中断
   - 每次中断扫描 1 位（灭位 -> 写段码 -> 开位）
   - 主循环每 300ms 更新显示数字

## 五、运行步骤

1. 在 Keil 新建 C51 工程，加入 `main.c`
2. 编译生成 `.hex`
3. 在 Proteus 完成连线并加载 `.hex`
4. 先跑静态模式，再切换动态模式
5. 如显示异常（全亮/全灭/乱码），优先检查两组极性宏

## 六、验收建议（按课程要求）

1. 静态显示：录制 0~9 循环现象
2. 动态显示：录制多位稳定显示且无明显闪烁
3. 报告中给出：原理图、代码、调试记录（极性调整过程）
