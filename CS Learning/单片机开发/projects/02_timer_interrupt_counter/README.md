# 实验 02：定时器中断 + 1s 计数闪烁

## 一、实验目标

掌握 51 单片机定时器 T0 的工作方式，使用 1ms 中断累计得到 1 秒节拍并翻转 LED。

## 二、硬件连接

- LED0 接 P3.0（低电平点亮）

## 三、功能要求

1. 配置 T0 为 16 位定时器模式（模式 1）
2. 以 12MHz 晶振为例，每 1ms 进入一次中断
3. 计满 1000 次中断后翻转 LED 状态

## 四、知识点

- TMOD、TH0、TL0、TR0、TF0
- ET0、EA 中断使能
- 中断服务函数写法
