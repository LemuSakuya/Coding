<h3 style="text-align: center;">实验项目名称：用户聚类分析</h3>

### 1. 实验背景
本次实验旨在对用户数据进行聚类分析，以便更好地了解用户的行为特征和消费模式。我们将使用K-Means聚类算法对数据进行分群，并通过性能指标评估聚类效果。

### 2. 数据加载与预处理
我们首先加载用户数据，并对其进行了标准化处理，以消除不同特征量纲的影响。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score, calinski_harabasz_score, davies_bouldin_score
from sklearn.preprocessing import StandardScaler

# 1. 数据加载
data = pd.read_csv('e:\\VSCode\\Coding\\Markdown\\机器学习笔记\\kmeans_data.csv', header=None)
data.columns = ['Feature1', 'Feature2']

# 2. 数据标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(data)
```

### 3. 确定最佳K值
为了确定最佳的聚类数量K，我们计算了不同K值下的轮廓系数（Silhouette Score）、Calinski-Harabasz指数和Davies-Bouldin指数。通过这些指标，我们可以选择最适合的K值。

```python
# 3. 确定最佳K值
range_n_clusters = range(2, 7)  # 增加K值范围到7
inertia = []
silhouette_scores = []

for n_clusters in range_n_clusters:
    kmeans = KMeans(n_clusters=n_clusters, random_state=42)
    cluster_labels = kmeans.fit_predict(X_scaled)
    inertia.append(kmeans.inertia_)
    silhouette_scores.append(silhouette_score(X_scaled, cluster_labels))
```

### 4. 应用K-Means
根据上述评估指标，我们选择K=4进行K-Means聚类，并将结果可视化。

```python
# 4. 应用K-Means (选择K=4)
kmeans = KMeans(n_clusters=4, random_state=42)
clusters = kmeans.fit_predict(X_scaled)
data['Cluster'] = clusters
centers = scaler.inverse_transform(kmeans.cluster_centers_)

# 5. 评估指标
silhouette = silhouette_score(X_scaled, clusters)
calinski = calinski_harabasz_score(X_scaled, clusters)
davies = davies_bouldin_score(X_scaled, clusters)

# 6. 结果可视化
plt.figure(figsize=(12, 5))
plt.subplot(1, 2, 1)
plt.scatter(data['Feature1'], data['Feature2'], c=data['Cluster'], cmap='viridis', alpha=0.7)
plt.scatter(centers[:, 0], centers[:, 1], c='red', marker='X', s=200, alpha=0.8)
plt.title('K-Means Clustering Results')
plt.xlabel('Feature1 (Purchase Amount)')
plt.ylabel('Feature2 (Browse Time)')
plt.colorbar(label='Cluster')

# 7. 聚类命名与解释
cluster_names = {
    0: "Low Value Users (Low Frequency, Low Amount)",
    1: "High Value Users (High Frequency, High Amount)",
    2: "Potential Users (Medium Consumption)",
    3: "Moderate Users"
}
data['Cluster_Name'] = data['Cluster'].map(cluster_names)

plt.subplot(1, 2, 2)
for cluster, name in cluster_names.items():
    cluster_data = data[data['Cluster'] == cluster]
    plt.scatter(cluster_data['Feature1'], cluster_data['Feature2'], 
                label=name, alpha=0.7)
plt.scatter(centers[:, 0], centers[:, 1], c='red', marker='X', s=200, alpha=0.8)
plt.title('Cluster Explanation')
plt.xlabel('Feature1 (Purchase Amount)')
plt.ylabel('Feature2 (Browse Time)')
plt.legend()

plt.tight_layout()
plt.show()
```

### 5. 聚类评估指标
我们使用轮廓系数（Silhouette Score）、Calinski-Harabasz指数和Davies-Bouldin指数来评估聚类效果。

```python
# 8. 输出结果
print("=== Clustering Evaluation Metrics ===")

print(f"Calinski-Harabasz Index: {calinski:.3f} (Higher is better)")
print(f"Davies-Bouldin Index: {davies:.3f} (Lower is better)")
```

### 8. 总结分析
通过K-Means聚类算法，我们将用户数据分为4个聚类，每个聚类代表一类用户的行为特征。评估指标显示聚类效果良好，轮廓系数接近1，Calinski-Harabasz指数较高，Davies-Bouldin指数较低。

- **轮廓系数(Silhouette Score)**：0.501，表明聚类效果较好，但仍有改进空间。
- **Calinski-Harabasz指数**：188.279，较高，表明聚类内部紧密且聚类之间较远。
- **Davies-Bouldin指数**：0.517，较低，表明聚类效果稳定。

通过可视化结果，我们可以更直观地看到每个聚类的分布情况和聚类中心的位置。未来，可以通过进一步调整K值和其他参数来优化聚类效果。
