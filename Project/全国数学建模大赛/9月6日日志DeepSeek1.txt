好的，我们来详细探讨一下作为非参数基准的**Kaplan-Meier（K-M）估计**。

这是一个在生存分析中至关重要、应用极其广泛的核心方法，常被称为**乘积极限法**。

---

### 1. 核心思想与目的

Kaplan-Meier估计的主要目的是**在存在“删失”数据的情况下，估计生存函数**。

*   **生存函数 S(t)**：描述个体存活时间超过时间点 t 的概率。即 S(t) = P(T > t)，其中 T 是生存时间。
*   **删失**：这是生存分析的关键特征。它指的是在研究结束时，某些个体的最终事件（如死亡、复发）尚未发生。我们只知道这些个体至少存活了到其最后一次被观察的时间。忽略删失或错误处理删失会导致严重偏差，而K-M方法能有效处理这一问题。

K-M估计通过**考虑每个唯一事件发生时间点**，以条件概率的方式，逐步计算出生存函数。

---

### 2. 计算方法与步骤

K-M生存曲线是一个右连续的阶梯函数，它在每个事件发生点下降。

**计算公式：**

$$
\hat{S}(t) = \prod_{t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)
$$

其中：
*   $\hat{S}(t)$：在时间 t 的生存概率估计值。
*   $t_i$：第 i 个事件发生的时间（按从小到大排序）。
*   $d_i$：在时间 $t_i$ 发生事件（如死亡）的个体数。
*   $n_i$：在时间 $t_i$ **之前仍处于风险中**的个体数（即在时间 $t_i$ 即将发生之前，仍被观察且未发生事件也未删失的个体数量）。
*   $\prod$：连乘符号，表示对所有小于等于 t 的事件时间点 $t_i$ 进行连乘。

**计算步骤：**
1.  **数据准备**：将所有个体的生存时间（包括发生事件的时间和删失时间）从小到大排序。
2.  **列出每个时间点**：对于每个发生事件的时间点 $t_i$（忽略仅有删失发生的时间点）。
3.  **计算风险集**：对于每个 $t_i$，确定此时处于风险中的个体数 $n_i$。
4.  **计算条件生存概率**：计算在时间 $t_i$ 之后仍然存活的**条件概率**：$(1 - \frac{d_i}{n_i})$。这个概率表示“对于一个已经存活到 $t_i$ 的个体，他能活过 $t_i$ 的概率”。
5.  **连乘**：将直到时间 t 的所有这些条件概率相乘，得到存活超过时间 t 的**无条件概率估计** $\hat{S}(t)$。

---

### 3. 一个简单示例

假设我们有一个小型研究，数据如下（“+”表示删失）：
| 患者 | 时间（月） | 事件（1=死亡，0=删失） |
| ：--- | :-------- | :-------------------- |
| A    | 2         | 1                     |
| B    | 3         | 0                     |
| C    | 5         | 1                     |
| D    | 6         | 1                     |
| E    | 8         | 0                     |
| F    | 10        | 1                     |

我们逐步计算K-M估计值：

| 时间 $t_i$ | 死亡数 $d_i$ | 风险集 $n_i$ | 条件概率 $(1 - d_i/n_i)$ | K-M生存率 $\hat{S}(t)$ |
| :--------- | :----------- | :----------- | :----------------------- | :--------------------- |
| 0          | -            | 6            | -                        | 1.000                  |
| 2          | 1            | 6            | $1 - 1/6 = 5/6$          | $1.000 \times 5/6 = 0.833$ |
| 5          | 1            | 4¹           | $1 - 1/4 = 3/4$          | $0.833 \times 3/4 = 0.625$ |
| 6          | 1            | 3²           | $1 - 1/3 = 2/3$          | $0.625 \times 2/3 \approx 0.417$ |
| 10         | 1            | 1³           | $1 - 1/1 = 0$            | $0.417 \times 0 = 0.000$   |

*¹ 在时间 t=5 之前，患者A（时间2死亡）和患者B（时间3删失）已经退出风险集，因此风险集 $n_i = 6 - 2 = 4$。*
*² 在时间 t=6 之前，患者C（时间5死亡）退出风险集，因此 $n_i = 4 - 1 = 3$。*
*³ 在时间 t=10 之前，患者D（时间6死亡）和患者E（时间8删失）退出风险集，因此 $n_i = 3 - 2 = 1$。*

**解读**：根据K-M曲线，我们可以说：
*   在2个月时，存活概率约为83.3%。
*   在5个月时，存活概率约为62.5%。
*   在6个月时，存活概率约为41.7%。
*   到10个月时，存活概率降至0%。

**注意**：删失数据（患者B和E）不直接贡献死亡事件，但它们会减少后续时间点的风险集人数 $n_i$，从而被巧妙地考虑进计算中。这正是K-M方法的精妙之处。

---

### 4. 为什么K-M是“非参数基准”？

1.  **非参数**：K-M估计不对生存时间的底层分布做任何假设（例如不假设它服从指数分布或Weibull分布）。它完全由数据驱动，让数据自己“说话”。这使得它非常灵活且适用性广。
2.  **基准**：
    *   **直观易懂**：其结果（生存曲线）非常直观，易于向临床医生或非统计学家解释。
    *   **稳健可靠**：由于其非参数特性，它在大多数情况下都能提供一个对真实生存函数的合理估计。
    *   **比较的基础**：**Log-Rank检验**通常用于比较两条或多条K-M曲线，这是组间生存差异比较最常用、最权威的非参数方法。
    *   **黄金标准**：在任何生存分析中，绘制K-M曲线并报告中位生存时间等都是首要且标准的步骤。任何更复杂的参数或半参数模型（如Cox比例风险模型）通常都需要与K-M的结果进行对照和验证。

---

### 5. 优点与局限性

**优点：**
*   有效处理**右删失**数据，这是生存分析中最常见的数据问题。
*   **无需分布假设**，适用性强。
*   计算简单，结果直观。
*   是组间比较的基石。

**局限性：**
*   当数据量很少时，估计的误差会较大（曲线末端尤其不稳定）。
*   它主要是一个**描述性工具**。虽然可以比较组间差异，但它本身并不能直接评估**协变量（如年龄、治疗方式）对风险的影响**，这是Cox模型更擅长的地方。
*   传统K-M通常只能处理一个分组变量（如治疗组 vs 对照组）。要同时考虑多个变量，需要借助其他模型。
*   对**时间依赖性变量**的处理能力较弱。

### 总结

**Kaplan-Meier估计是生存分析领域不可或缺的非参数基准方法。** 它提供了一种简单、稳健、无需假设的方式来描述生存数据，尤其是在存在删失的情况下。几乎所有涉及时间至事件数据的临床研究、工程可靠性研究等领域，K-M曲线都是报告结果的首要工具。尽管有其局限性，但它作为探索性分析和结果展示的第一步，地位无可替代。

def create_advanced_km_plot_quartiles(data):
    try:
        # 数据预处理和类型转换
        data = data.copy()
        data['time_to_event'] = pd.to_numeric(data['time_to_event'], errors='coerce')
        data['event_occurred'] = pd.to_numeric(data['event_occurred'], errors='coerce')
        
        # 移除无效数据
        data_clean = data.dropna(subset=['time_to_event', 'event_occurred', 'quartile_group'])
        
        # 确认分组变量是分类类型
        data_clean['quartile_group'] = data_clean['quartile_group'].astype(str)
        
        # 获取分组
        groups = data_clean['quartile_group'].unique()
        groups.sort()
        
        # 初始化图表
        plt.figure(figsize=(12, 8))
        ax = plt.subplot(111)
        
        # 为每个分组绘制KM曲线
        for i, group in enumerate(groups):
            group_data = data_clean[data_clean['quartile_group'] == group]
            kmf = KaplanMeierFitter()
            kmf.fit(
                durations=group_data['time_to_event'],
                event_observed=group_data['event_occurred'],
                label=f'Quartile {group} (n={len(group_data)})'
            )
            kmf.plot_survival_function(ax=ax, ci_show=False)
        
        # 执行Log-Rank检验
        try:
            results = logrank_test(
                data_clean['time_to_event'].values,
                data_clean[data_clean['quartile_group'] == groups[0]]['time_to_event'].values,
                event_observed_A=data_clean['event_occurred'].values,
                event_observed_B=data_clean[data_clean['quartile_group'] == groups[0]]['event_occurred'].values,
                labels=groups
            )
            
            # 添加统计检验结果到图表
            plt.text(0.05, 0.05, f'Log-Rank Test\np-value: {results.p_value:.4f}', 
                    transform=ax.transAxes, fontsize=12, bbox=dict(boxstyle="round,pad=0.3", 
                    facecolor="lightgray", alpha=0.7))
            
        except Exception as e:
            print(f"Log-Rank test failed: {e}")
            # 可以继续绘制图表，只是不添加统计结果
        
        # 设置图表属性
        plt.title('Kaplan-Meier Survival Curves by Quartile Groups')
        plt.xlabel('Time')
        plt.ylabel('Survival Probability')
        plt.grid(True, alpha=0.3)
        plt.legend()
        
        return plt.gcf()
        
    except Exception as e:
        print(f"Error in creating KM plot: {e}")
        return None
